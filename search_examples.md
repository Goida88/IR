# Примеры запросов и недостатки выдачи

Примеры запросов:
- trigeminal neuralgia treatment
- basal ganglia function
- dopamine receptor agonist
- multiple sclerosis symptoms
- нейропластичность механизмы
- антидепрессанты СИОЗС побочные эффекты
- расстройства сна лечение

Недостатки:
- Google site: может подмешивать нерелевантные страницы (служебные, разные языковые версии, не относящиеся к выбранной тематике).
- Поиск Википедии чувствителен к формулировкам и синонимам, ранжирование не всегда оптимально для задач информационного поиска.
- Нет гибкой настройки: поиск только по телу/только по заголовкам и т.п. в стандартной выдаче.
