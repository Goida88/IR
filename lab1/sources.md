# Источники данных

Корпус собран из двух источников:

1) English Wikipedia (en.wikipedia.org)
2) Russian Wikipedia (ru.wikipedia.org)

Способ получения: MediaWiki API (w/api.php), выгрузка страниц по тематическим категориям, сохранение:
- raw: wikitext (вики-разметка)
- text: извлечённый plain text

Поиск по исходным документам:
- встроенный поиск Википедии (en/ru)
- Google с ограничениями site:en.wikipedia.org и site:ru.wikipedia.org
