# Лабораторные работы по информационному поиску

Проект с выполненными лабораторными работами по курсу «Информационный поиск» (МАИ).

Выполнил: студент Кузнецов М.С. М8О-401Б-22

Корпус: Wikipedia **EN+RU** по тематике **биомедицины и психического здоровья**, объём **30 000** документов.

## Реализованные работы
- **ЛР1 — Добыча корпуса документов**  
  Сбор статей из MediaWiki API. Хранение raw (wikitext) + извлечённый текст + metadata.

- **ЛР2 — Поисковый робот**  
  Робот для обкачки URL с сохранением результатов в MongoDB: нормализация URL, сохранение сырого HTML, отметка времени (Unix timestamp), корректное продолжение после остановки и переобкачка при изменениях.

- **ЛР3 — Токенизация**  
  Токенизатор на C++ для EN/RU текста в UTF-8. Выдаёт статистику по количеству токенов, средней длине токена и скорости обработки.

- **ЛР4 — Стемминг**  
  Нормализация токенов для уменьшения вариативности словоформ перед индексированием.

- **ЛР5 — Закон Ципфа**  
  Подсчёт частот терминов по корпусу, построение распределения в логарифмической шкале и наложение аппроксимации законом Ципфа (оценка параметра *s*).

- **ЛР7 — Булев индекс**  
  Инвертированный индекс вида `term → postings (список docID)`, хранение на диске в бинарном формате + словарь терминов.

- **ЛР8 — Булев поиск**  
  Выполнение булевых запросов **AND / OR / NOT** со скобками и приоритетами. Интерфейс представлен в двух вариантах: CLI и веб-страница (форма ввода + выдача результатов).
